เชื่อมต่อ Google Sheet

ข้อมูลทั้งหมดจะเก็บใน Google Sheet ของคุณเอง ทำครั้งเดียวก็ใช้ได้เลย
1 สร้าง Google Sheet ใหม่
เปิด sheets.new เพื่อสร้าง Google Sheet ใหม่ ตั้งชื่อว่า "บัญชีคู่" แล้วเปิดทิ้งไว้
2 เพิ่ม Apps Script
ใน Google Sheet → เมนู Extensions → Apps Script → ลบโค้ดเดิมออก → วางโค้ดด้านล่างนี้แทน → กด Save
3 Deploy เป็น Web App
ใน Apps Script Editor → Deploy → New deployment → เลือก Type: Web app → Execute as: Me → Who has access: Anyone → กด Deploy → Copy URL ที่ได้มาวางด้านล่างนี้
4 วาง Web App URL
⚠️ URL นี้คือ API ของคุณ — ไม่ควรแชร์ให้คนอื่นที่ไม่ใช่แฟน
มี URL อยู่แล้ว? → กรอก URL แล้วข้ามไปเลย
